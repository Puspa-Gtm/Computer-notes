Introduction to the Internet

The Internet is a global network of interconnected computers and devices that enables communication and data exchange. It has transformed how we access information, communicate, and perform various tasks. Initially developed as a research project, the Internet has grown into an essential part of modern life, supporting education, business, social interaction, and entertainment.

Evolution of the Internet

The Internet originated from ARPANET, a project by the United States Department of Defense in the 1960s. Over the decades, it evolved with the introduction of protocols such as TCP/IP, making it more reliable and scalable. By the 1990s, the World Wide Web (WWW) became a popular platform for browsing and sharing information.

Major Uses of the Internet

The Internet serves various purposes, including:

  • Communication: Email, instant messaging, video calls.
  • Information Retrieval: Accessing websites, search engines, and online libraries.
  • Social Networking: Platforms like Facebook, Twitter, and Instagram.
  • E-commerce: Online shopping and digital transactions.
  • Education: Online learning, courses, and digital resources.

Internet Services and Applications

World Wide Web (WWW)

The World Wide Web (WWW) is an interconnected system of public web pages accessible through the internet. It uses the Hypertext Transfer Protocol (HTTP) to transmit data and allows users to view web pages and navigate between them using hyperlinks. The WWW was created in 1990 at the European Particle Physics Laboratory (CERN) in Geneva, Switzerland.

Uniform Resource Locator (URL)

A URL (Uniform Resource Locator) is a unique address used to identify the location of a resource on the internet. URLs are used by web browsers to retrieve web pages, images, and other content from servers. A URL typically consists of a protocol (such as HTTP or HTTPS), a domain name (like www.example.com), and an optional path to a specific file or page.

HTTPS (Hypertext Transfer Protocol Secure)

HTTPS is an extension of HTTP that provides secure communication over a computer network. It encrypts data between the web browser and server to ensure privacy and data integrity. Websites that handle sensitive information, such as online banking, use HTTPS to protect user data.

Web Pages and Websites

A web page is a document on the World Wide Web that is typically written in HTML and viewed through a web browser. It can contain text, images, videos, and interactive content. A website is a collection of related web pages hosted on a server and identified by a common domain name.

FTP (File Transfer Protocol)

FTP (File Transfer Protocol) is a standard network protocol used to transfer files from one host to another over the internet. It allows users to upload, download, and manage files on a remote server. FTP is commonly used for uploading website files to hosting servers.

Web Browser

A web browser is a software application used to access and view web pages on the internet. It retrieves content from web servers and displays it on the user's screen. Popular web browsers include Google Chrome, Mozilla Firefox, Safari, and Microsoft Edge.

Common Terms Related to the Internet

1. ISP (Internet Service Provider)

An Internet Service Provider (ISP) is an organization that provides Internet access to users. ISPs offer services such as broadband, fiber optics, and dial-up connections. They act as intermediaries between users and the Internet, allowing them to connect to the global network. Some well-known ISPs in Nepal are Mercantile, Everest Net, and WorldLink.

2. Link

A link, also known as a hyperlink, is a clickable element on a web page that directs users to another page or resource. Links can be text or images and are typically highlighted or underlined to indicate interactivity. They serve as an essential navigation tool, allowing users to move between web pages and access related information.

3. Search Engine

A search engine is a software system that searches the Internet for specific information based on user queries. It indexes websites and web pages to provide relevant search results. Popular search engines include Google, Bing, Yahoo, and DuckDuckGo. They help users find information quickly and efficiently by ranking web pages according to relevance and content quality.

4. Domain Name and IP Address

A domain name is a human-readable address used to access websites on the Internet. It typically consists of two or more parts separated by dots, such as 'example.com.' Each domain name corresponds to a unique IP (Internet Protocol) address, which identifies the server hosting the website. IP addresses are numerical labels that facilitate data transfer between devices over the network.

5. DNS Server

A DNS (Domain Name System) server translates domain names into IP addresses, allowing users to access websites using human-readable URLs instead of numerical addresses. DNS servers maintain a database of domain name mappings and respond to queries by providing the corresponding IP address. This system simplifies navigation and enhances user convenience.

6. Web Server

A web server is a computer system that hosts websites and delivers web pages to users upon request. It processes incoming HTTP or HTTPS requests from clients (web browsers) and serves the requested content. Web servers play a crucial role in maintaining website availability and performance. Apache, Nginx, and Microsoft IIS are some of the most widely used web server software.

Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages and applications. It provides the basic structure of a webpage and allows users to incorporate text, images, videos, links, and other multimedia elements. HTML files are saved with a ".html" or ".htm" extension and are interpreted by web browsers to display content.

Uses of HTML

  • Creating the basic structure and layout of a webpage.
  • Embedding multimedia elements like images, videos, and audio files.
  • Providing interactive features through forms and links.
  • Formatting text with headings, paragraphs, lists, and more.
  • Enabling cross-browser compatibility to ensure consistent display.

HTML Attributes

HTML attributes are additional information provided within HTML tags to define properties or modify behavior. Attributes are included within the opening tag and usually consist of a name and value pair, separated by an equals sign. For example:

        <a href="https://www.example.com">Visit Example</a>

Common attributes include:

  • href: Specifies the URL of a link.
  • src: Defines the source file for images or other media.
  • alt: Provides alternative text for images.
  • title: Adds a tooltip to an element.
  • style: Applies inline CSS styles to elements.

Introduction to CSS

CSS (Cascading Style Sheets) is a style sheet language used to define the presentation of a document written in HTML or XML. It controls the layout, color schemes, font styles, spacing, and overall visual appearance of web pages. CSS separates content from design, making it easier to maintain and update the presentation without altering the underlying structure. CSS is essential for creating visually appealing, responsive, and user-friendly websites.

Types of CSS

  • Internal CSS

    Internal CSS is used to define styles within the <style> tag located inside the <head> section of an HTML document. This method is useful when applying styles to a single page. Although it keeps the CSS centralized, it can make the document lengthy and less manageable when dealing with extensive styling.

  • Inline CSS

    Inline CSS is written directly within the HTML element using the style attribute. It is useful for applying unique styles to individual elements. However, it is generally discouraged because it mixes content with presentation, making the code less organized and harder to maintain.

  • External CSS

    External CSS involves linking an HTML document to an external stylesheet file using the <link> tag within the <head> section. It allows for the separation of style from content, promoting cleaner and more maintainable code. External stylesheets can be reused across multiple pages, reducing redundancy and ensuring a consistent design.
